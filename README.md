# Online Examination System

This project is a full-stack Online Examination System designed to conduct exams in a simple and efficient way. It allows students to take exams online with a timer-based interface, while teachers/admins can manage questions and users.

---

##  Features

-  Student & Teacher Login
-  Student Dashboard with subject selection
-  Timed Examination Interface
-  Multiple Choice Questions (MCQs)
-  Question Navigation (Next/Previous)
-  Teacher Panel for adding questions
-  Admin panel for managing users
-  Responsive Design

##  Installation & Setup

### 1️ Clone the Repository
git clone https://github.com/your-username/online-examination-system.git
cd Online_Examination_system-main

---

### 2️ Setup Frontend
cd frontend
npm install
npm run dev

---

### 3️ Setup Backend
cd backend
mvn spring-boot:run

---

### 4️ Database Setup
- Configure MySQL in application.properties
- Run required SQL scripts

---

##  How It Works

1. User logs in as Student or Teacher  
2. Student selects subject and starts exam  
3. Questions are displayed one by one  
4. Timer tracks exam duration  
5. Student submits answers  
6. Teacher can add/manage questions
